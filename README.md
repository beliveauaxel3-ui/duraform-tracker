# Duraform Tracker — Fondations Béliveau inc.

## Déploiement sur Vercel

1. Crée un compte sur https://github.com (si tu n'en as pas déjà un)
2. Crée un nouveau repository (ex: "duraform-tracker")
3. Glisse-dépose TOUS les fichiers de ce dossier dans le repository GitHub
   (garde la structure: package.json, vite.config.js, index.html, et le dossier src/ avec main.jsx et App.jsx)
4. Va sur https://vercel.com et connecte-toi avec ton compte GitHub
5. Clique "Add New Project", choisis ton repository "duraform-tracker"
6. Clique "Deploy" — Vercel détecte automatiquement que c'est une app Vite/React
7. En ~60 secondes, ton app est en ligne à une adresse comme: duraform-tracker.vercel.app

## Pour faire des mises à jour plus tard

1. Demande à Claude de modifier le fichier App.jsx
2. Remplace l'ancien App.jsx sur GitHub par le nouveau (glisser-déposer dans le dossier src/ sur GitHub)
3. Vercel redéploie automatiquement en ~30 secondes
